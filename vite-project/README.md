# Pokémon Universe

A beautiful Pokémon fan website built with React + Vite + Framer Motion.

## Features
- Animated hero section with draggable Pokémon strip
- Pokédex slider with stats, types, and descriptions  
- Pokémon types section
- Full sign-up / login / sign-out auth
- Mobile responsive

## Setup

```bash
npm install
npm run dev
```

## Assets
See `ASSETS_GUIDE.html` for instructions on adding your Pokémon images.
Place images in `public/assets/hero/` and `public/assets/slider/`.

## Build
```bash
npm run build
npm run preview
```
